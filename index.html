AI-Chatbot for Discord

The bot listens and responds. It can !ask, !draw, !edit, or type "prompt" after !ask for an image prompt. It has a Draw button! The bot operates in Python via the discord.py library. It accesses the OpenAI API to process text, while drawing also can be done via the same API, or choose the local branch on GitHub.

Cat Tower Value

Cat real estate prices are through the roof. Cat towers are complex and expensive, but this cat-ulator can handle it. It takes all the cat tower's shapes and uses the price to give the actual dollar per square inch. It works for humans. Let your feline give it a whirl.
